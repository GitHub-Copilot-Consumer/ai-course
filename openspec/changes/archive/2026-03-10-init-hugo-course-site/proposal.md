## Why

We need a maintainable course website that can be versioned with the rest of the materials and published quickly. Hugo provides a fast, static-site workflow that fits course content updates.

## What Changes

- Add a Hugo site scaffold for the course website.
- Define the initial content structure for lessons, assignments, and resources.
- Include baseline configuration, theme choice, and build/deploy scripts.

## Capabilities

### New Capabilities
- `hugo-course-site`: Provide a Hugo-based course site with a defined content structure and build pipeline.

### Modified Capabilities
- None.

## Impact

- Adds Hugo configuration and content directories under the repository.
- Introduces a theme (module or submodule) and site assets.
- Adds scripts or documentation for building and publishing the site.
