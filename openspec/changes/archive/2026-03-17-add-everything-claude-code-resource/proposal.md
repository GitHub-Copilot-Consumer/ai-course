## Why

課程 resources 目前僅有 OPSX 指令速查，缺乏對主要 AI 開發工具生態資源的介紹。`everything-claude-code` 是一個廣受社群使用的 agent harness 系統，同時支援 OpenCode，加入此資源說明可幫助學員了解業界實際工具鏈，並掌握如何與 OpenCode 整合使用。

## What Changes

- 在 `site/content/resources/` 新增 `everything-claude-code.md` 頁面
- 說明 `everything-claude-code` 是什麼、核心功能（Skills、Hooks、Commands、Rules、Instincts）
- 說明與 OpenCode 的整合方式（Plugin 安裝、手動安裝、實際使用場景）
- 更新 `_index.md` 或導覽以包含此新頁面（如有需要）
